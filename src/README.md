# Source Folder Guide

Use this as the quick map when you need to edit the app.

## app

- `App.jsx` - main app shell, login/dashboard switch, logout flow.
- `main.jsx` - Vite/React entry file and global style imports.

## assets

- `branding/` - app logos and brand images.
- `icons/` - module or feature images.

## features

- `dashboard/` - dashboard page, sidebar navigation, module workspace shell.
- `expenses/` - daily expense manager and daily expense entry UI.
- `login/` - login page, login form, login layout, and module list data.

## shared

- `services/` - reusable browser/API helper services, such as auth storage.
- `styles/` - global CSS used by the whole app.

## Build Output

- `dist/` is generated by `npm run build`.
- `node_modules/` is installed by `npm install`.
